# Zone Wars — Badge Design Document

## Style Prompt (use as prefix for every badge icon)

```
2D illustrated game icon, 512x512, hand-painted cartoon style, soft shading with bold colors,
thick clean outlines, highly saturated vibrant palette, slight painterly texture, soft gradient
background (warm pastel tones), clean and readable at small sizes, exaggerated proportions,
mobile game UI art style, Roblox simulator shop icon aesthetic, no text
```

## How to Use

For each badge, combine the **style prompt** + the **subject prompt**:

```
{style prompt}, {subject prompt}
```

Example for **Settler**:
```
3D rendered Roblox-style game icon, 512x512, chunky cartoon aesthetic, soft rounded shapes,
thick bold outlines, highly saturated vibrant colors, smooth plastic-like material, soft gradient
background (light blue to white), subtle drop shadow, clean and readable at small sizes,
toy-like proportions, glossy highlights, Roblox simulator game art style, no text,
Small green polygon land plot on a grass patch, tiny wooden flag planted in center, beginner tier, green and brown tones
```
